# Установка

Для работы с Joke на вашей машине должны быть установлены:

- PHP 8.4 или новее
- Composer — менеджер зависимостей для PHP [Composer](https://getcomposer.org/)

> Примечание: На момент написания документации фреймворк находится в стадии подготовки к первому релизу. Установка
> возможна только через Composer.

## Быстрый старт

1. Создайте новый проект (или используйте существующий):
    ```bash
    composer init
    ```
2. Установите ядро фреймворка:

    ```bash
    composer require voral/joke
    ```

3. Настройте точку входа.

   Убедитесь, что корневая директория веб-сервера указывает на папку public/. Например, при использовании встроенного
   сервера PHP:

    ```bash
    php -S localhost:8000 -t public/
    ```

Теперь ваше приложение доступно по адресу http://localhost:8000.
